##  Challenge Vuejs Restaurant

### features
  - dark mode 
  - search 
  - rate limiting / storage
  - Unit Tests / Integration Tests
  - Joke

### optimizations
  - lazyloading 
  - code-spliting 
  - tree shaking

### Commands
  - to start = ```$ npm i && npm run serve```
  - unit test = ```$ jest```
  - end to end tests = ```$ (npm bin)/cypress open```
  - build = ```$ npm run build-report```
